# 升级日志

本文件用于记录 `MyPassword` 每一次 Release 包的更新内容。

建议规则：
- 每次生成新的测试包或正式发布包前，新增一个版本区块。
- 版本号建议使用 `主版本.次版本.修订号`，例如 `v0.4.0`。
- 日期统一使用 `YYYY-MM-DD` 格式。
- 内容按 `新增`、`优化`、`修复`、`已知问题` 记录，便于测试和回溯。
- 如果该版本对应有安装包，建议补充安装包文件名或内部构建编号。

推荐模板：

```md
## v0.0.0 - 2026-04-22

### 新增
- 新功能 1
- 新功能 2

### 优化
- 优化项 1

### 修复
- 修复项 1
- 修复项 2

### 已知问题
- 已知问题 1
```

## v0.1.2 - 2026-04-23

### 发布信息
- 安装包：`releases/MyPassword_release_0.1.2.apk`
- 内部版本：`0.1.2+3`

### 新增
- 新增同步冲突处理页，可查看本地侧与远端侧状态，并直接选择“保留本地并上传”或“保留远端并恢复”。

### 优化
- 首页和同步状态页在检测到同步冲突时，可直接跳转到冲突处理页，而不是只停留在提示文案。
- 抽离 WebDAV 上传与恢复公共处理逻辑，统一同步相关交互和提示。

### 修复
- 修复同步冲突处理完成后仍可能重复提示同一远端版本的问题，现已记录“已处理远端版本”状态。

### 已知问题
- 自动同步仍为单向上传策略，当前版本尚不支持本地与远端的自动合并。
- 通过生物识别解锁进入应用后，由于当前会话中没有输入主密码，自动上传不会执行；此时仍可手动同步。

## v0.1.1 - 2026-04-22

### 发布信息
- 安装包：`releases/MyPassword_release_0.1.1.apk`
- 内部版本：`0.1.1+2`

### 新增
- 新增设置页一级分组结构，将“本地备份”和“WebDAV 同步”分别归类。
- 新增 WebDAV 上传与恢复过程中的处理中弹窗，提升同步过程可感知性。

### 优化
- WebDAV 测试连接成功后会自动保存当前配置，避免返回后配置丢失。
- 调整设置页最终顺序，统一为本地备份、WebDAV 同步、主密码、语言、自动同步、锁定时机、生物识别等结构。

### 修复
- 修复 release 包缺少 `INTERNET` 权限，导致 WebDAV 在测试包中无法联网的问题。
- 修复 WebDAV 远端路径以 `/` 开头时会丢失 `/dav/` 前缀的问题。
- 修复上传到 WebDAV 时远端目录不存在导致失败的问题，现会自动创建远端目录。
- 修复“测试 WebDAV 连接”只校验不保存配置的问题。
- 修复“立即锁定保险库”在错误锁定模式下仍然显示的问题。

### 已知问题
- 当前 `releases/` 目录内已存在旧的 `0.1.0` 安装包，本版本开始按新版本号单独归档。
- 自动同步的结果状态页和冲突处理页仍未完成。

## v0.4.0 - 2026-04-22

### 新增
- 新增 WebDAV 云同步第一期能力，可配置用户自己的 WebDAV 网盘用于同步加密同步包。
- 新增 WebDAV 配置页面，支持填写 WebDAV 地址、用户名、应用密码、远端同步路径。
- 新增 WebDAV 连接测试能力，可在保存配置前验证连接是否可用。
- 新增手动上传到 WebDAV 功能。
- 新增从 WebDAV 恢复同步包功能。
- 新增“本地变更后自动同步”设置开关。
- 新增首页远端同步包更新提示，当 WebDAV 上有较新的同步包时可提示用户处理。

### 优化
- 同步配置中的 WebDAV 应用密码改为保存在安全存储中，不再保存在普通偏好设置中。
- 自动同步策略接入新增、编辑、删除、导入备份、修改主密码等本地数据变更场景。
- 自动同步采用“后台尽力同步”策略，不阻塞本地保存和删除操作。
- 同步状态增加最近本地变更、最近上传、最近下载、最近错误记录，便于后续扩展同步状态展示。

### 修复
- 细化 WebDAV 同步错误提示，支持区分以下场景：
- WebDAV 未配置
- WebDAV 认证失败
- 网络连接失败
- 远端同步包不存在
- WebDAV 配置无效
- 当前主密码错误
- 备份文件主密码错误

### 已知问题
- 通过生物识别解锁进入应用后，由于当前会话中没有输入主密码，自动上传不会执行；此时仍可手动同步。
- 当前版本暂不支持自动合并远端与本地数据，发现远端更新时仍需用户手动确认并执行恢复。

## v0.3.0 - 2026-04-22

### 新增
- 新增设置项“锁定保险库时机”，支持：
- 应用切到后台时立即锁定
- 仅在应用进程结束后锁定
- 新增修改主密码功能。
- 新增导出目录设置，可在设置一级菜单中修改，并持续生效直到下次修改。
- 新增导入备份时的备份文件主密码校验流程。
- 新增导出备份时的当前主密码校验流程。

### 优化
- 调整设置页菜单顺序，便于用户先导入、导出，再配置导出目录。
- 导入备份流程调整为：
- 选择备份文件
- 输入备份文件主密码
- 校验通过后再执行导入
- 若当前无保险库，则导入前先创建新保险库
- 导出备份流程调整为：
- 输入当前应用主密码
- 生成加密备份文件
- 使用已配置的导出目录输出文件

### 修复
- 修复导出目录此前放在导出流程内部、导致每次都要重新选择目录的问题。
- 修复解锁页生物识别提示文案中英文混杂的问题，统一跟随应用语言。
- 修复导入备份相关文案不清晰的问题。
- 修复导入备份时点击选择文件无响应的问题，改为走原生文件选择能力。
- 修复备份导入时主密码错误提示不准确的问题，错误时提示“备份文件主密码错误”。

### 已知问题
- 导入备份遇到重复项时，当前按“标题 + 用户名 + 域名”判定重复并跳过导入，暂不支持用户手动选择覆盖或保留两份。

## v0.2.0 - 2026-04-22

### 新增
- 新增条目列表搜索框，可按名称搜索密码条目。
- 新增条目创建能力，支持填写名称、账号、密码、备注、网站、收藏状态。
- 新增系统生成密码功能。
- 新增条目详情页“复制账号”“复制密码”操作。
- 新增收藏能力，收藏条目默认显示在列表前面。
- 新增列表左滑删除能力，并增加二次确认弹窗。

### 优化
- 默认进入系统后展示条目列表，不再先进入其他中间页面。
- 列表中同时显示条目名称和账号信息，便于快速识别。
- 详情页 UI 重新设计，和全局视觉风格保持一致。
- 列表左滑时无需滑到底，露出删除按钮即可点击。
- 左滑后不立即弹窗，改为点击删除按钮后再进行二次确认。

### 修复
- 修复首次创建或导入保险库后，再次进入应用未正确回到密码解锁页的问题。
- 修复创建保险库或导入备份后，后续启动流程未进入主密码解锁页的问题。
- 修复新增、查看、编辑条目主流程不完整的问题。

### 已知问题
- 当前删除仍为逻辑删除策略，后续如有彻底物理删除需求，需要单独补充设计。
